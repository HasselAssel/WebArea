Information

WebArea lets you turn a website into a link. The whole website lives inside the URL. There is limited space in the URL, so try to avoid image (or large file) uploads and use links for resources in your code.

The default file loaded is 'index.html'. If this file doesn't exist or one of the query parameters '?f' or '?d' is added to the URL, the files will just be listed and not displayed (this link can be copied from the button on the right).
